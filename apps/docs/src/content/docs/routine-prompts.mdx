# Routine Prompts

Use these prompts with **Run now** or `pnpm routine:fire -- "...prompt..."`.

## Primary Demo

```text
Analyze issue PROD-4521.
Use only this repo.
Write reports/triage-PROD-4521.md.
Cite exact file paths.
Explain whether hotfix or rollback is safer.
Include Decision, Severity, Likely Cause, Evidence, Immediate Actions, Customer/Internal Update, Confidence, and Missing Data.
```

Expected evidence:

- `issues/PROD-4521.json`
- `deploys/recent.json`
- `runbooks/payment-service-degraded.md`
